# Regex Walkthrough

## Setup

1. Install dependencies using `uv`:
   ```bash
   uv sync
   ```

2. When running the notebooks in VS Code, select the `.venv` Python interpreter from the kernel selector.

## Activities

### Regex Treasure Hunt
Start with **Regex_Treasurehunt.ipynb** to explore regex patterns through guided exercises.

### Extra Practice
If you finish early, work through **regex-practice.ipynb** for additional regex challenges.
